Get Started With Appverse

Appverse Web provides an incredibly powerful and flexible collection of technologies to improve your enterprise Application.

Start From a Maven Archetype

Create maven projects from arquetipes executing thefollowing commands:

GWT Flavored Project

mvn archetype:generate -DarchetypeGroupId=org.appverse.web.framework.archetypes.gwt -DarchetypeArtifactId=appverse-web-archetypes-gwt -DarchetypeVersion=1.1.0-RELEASE -DgroupId=org.organization.project -DartifactId=project -Dversion=1.0-SNAPSHOT "-DnewProjectName=My Project" -DtargetDbPlatform=MysqlPlatform -DinteractiveMode=false
  • archetypeGroupId org.appverse.web.framework.gwt
  • archetypeArtifactId appverse-web-archetypes-gwt
  • archetypeVersionId1.1.0-RELEASE
  • groupId New project groupId e.g. org.organization.project*
  • artifactId New project artifactId
  • version New project version eg.1.0-SNAPSHOT
  • newProjectName New project user-friendly name eg.1.0-SNAPSHOT
  • targetDbPlatform Valid eclipselink database platform e.g. OraclePlatform (Defaults to MySQLPlatform)
* GroupId must finish with artifactId

JSF2 Flavored Project

mvn archetype:generate -DarchetypeGroupId=org.appverse.web.framework.archetypes.jsf2 -DarchetypeArtifactId=appverse-web-archetypes-jsf2 -DarchetypeVersion=1.1.0-RELEASE -DgroupId=org.organization.project -DartifactId=project -Dversion=1.0-SNAPSHOT "-DnewProjectName=MyProject" -DinteractiveMode=false
  • archetypeGroupId org.appverse.web.framework.jsf2
  • archetypeArtifactId appverse-web-archetypes-jsf2
  • archetypeVersionId1.1.0-RELEASE
  • groupId New project groupId e.g. org.organization.project*
  • artifactId New project artifactId
  • version New project version eg.1.0-SNAPSHOT
  • newProjectName New project user-friendly name eg.1.0-SNAPSHOT
* GroupId must finish with artifactId
