1.Abstract

START FROM MAVEN ARCHETYPE

Create maven projects from arquetipes executing the following commands:

GWT

mvn archetype:generate -DarchetypeGroupId=org.appverse.web.framework.archetypes.gwt -DarchetypeArtifactId=appverse-web-archetypes-gwt -DarchetypeVersion=1.1 -DgroupId= org.organization.project -DartifactId=project -Dversion=1.0-SNAPSHOT -DnewProjectName=New Project -DtargetDbPlatform=OraclePlatform -DinteractiveMode=false

·         archetypeGroupId           org.appverse.web.framework.gwt

·         archetypeArtifactId         appverse-web-archetypes-gwt

·         archetypeVersionId        1.1

·         groupId                                           New project groupId  e.g. org.organization.project*

·         artifactId                                         TNew project artifactId  eg. project*

·         version                                             New project version  eg. 1.0-SNAPSHOT

·         newProjectName                    New project user-friendly name  e.g New Project

·         targetDbPlatform                  Valid eclipselink database platform e.g. OraclePlatform (Defaults to

                                                                MySQLPlatform)

* groupId must finish with artifactId

JSF2

mvn archetype:generate -DarchetypeGroupId=org.appverse.web.framework.archetypes.jsf2 -DarchetypeArtifactId=appverse-web-archetypes-jsf2 -DarchetypeVersion=1.1 -DgroupId=org.organization.project -DartifactId=project -Dversion=1.0-SNAPSHOT -DnewProjectName=New Project -DinteractiveMode=false

·         archetypeGroupId           org.appverse.web.framework.gwt

·         archetypeArtifactId         appverse-web-archetypes-gwt

·         archetypeVersionId        1.1

·         groupId                                           New project groupId  e.g. org.organization.project*

·         artifactId                                         TNew project artifactId  eg. project*

·         version                                             New project version  eg. 1.0-SNAPSHOT

·         newProjectName                    New project user-friendly name  e.g New Project

* groupId must finish with artifactId
